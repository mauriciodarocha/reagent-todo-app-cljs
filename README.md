
# Todo app in ClojureScript + Reagent

## Intallation

```
npm install
```

## Running development environment

```
npm run dev
```

or run

```
npx shadow-cljs watch app
```

### Start a ClojureScript REPL

```
npx shadow-cljs browser-repl
```

## Building for production

```
npx shadow-cljs release app
```
